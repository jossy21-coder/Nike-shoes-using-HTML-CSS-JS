# 👟 Nike Shoes Website

A modern and responsive Nike shoes landing page built using **HTML** and **CSS**. This project focuses on clean UI design, smooth animations, and interactive elements to create an engaging user experience.

---

## 📌 Features

* 🔥 Clean and modern user interface
* 📱 Fully responsive design (mobile, tablet, desktop)
* 🎨 Smooth CSS animations and transitions
* 🖼️ Interactive product showcase
* 🛒 Add-to-cart interface (UI only)
* ⚡ Fast and lightweight performance

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling, Flexbox & Grid layout

---

## ⚙️ How to Run the Project

1. Download or clone this repository:

   ```
   git clone https://github.com/jossy21-coder/Nike-shoes.git
   ```

2. Open the project folder:

   ```
   cd Nike-shoes
   ```

3. Run the project:

   * Open `index.html` in any web browser

---

## 💡 Project Overview

This project displays Nike shoes in a visually appealing layout. Users can explore products through interactive UI elements such as buttons, hover effects, and animations. It is designed as a frontend-only project for practice and portfolio use.
